# Teste OM30

Projeto desenvolvido usando vue3 + tailwindcss + miraje.js


## Execução Projeto

```sh
npm install
```
### Copiar o .env

```sh
windows
copy .env.example .env

linux
cp .env.example .env
```

### Compilar e rodar Hot-reload

```sh
npm run dev
```
